# Run Tests Command

Execute the full test suite for {{PROJECT_NAME}}.

## Steps

1. Run unit tests
2. Run integration tests
3. Check code coverage
4. Report results

## Commands

```bash

```

## On Failure

If any tests fail:
1. Read the error message carefully
2. Identify the failing test
3. Check the related code
4. Attempt to fix the issue
5. Re-run the tests
